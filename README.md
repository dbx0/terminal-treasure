# Terminal Treasure

A terminal-based idle RPG game.

## Features

- Terminal-based gameplay using blessed
- Multiple currencies to collect and upgrade
- Save and load game progress
- Particle effects and animations

## Requirements

- Python 3.x
- blessed

## Installation

1. Clone the repository
2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the game:
```bash
python run.py
```

Run with debug mode:
```bash
python run.py --debug
```

## Game Controls

- Use arrow keys to navigate the menu
- Press Enter to select
- Press Q or Escape to exit
- Press C to sell all the items you collected
- Press M to add coins
- Press I to hide/show inventory
